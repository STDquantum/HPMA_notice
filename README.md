[subtitle2]咒语卡调整[-]



1.光辉咒语卡“万咒皆终”



每次使用“万咒皆终”时，其衍生卡“魔杖”的魔力值消耗都将在上一次使用“万咒皆终”时“魔杖”的魔力值基础上增加1点。同时，来自回响效果释放的“万咒皆终”将能够获得“魔杖”。



2.传说咒语卡“标准咒语书”



“标准咒语书”的魔力值消耗增加至5，该调整不影响提供的5张一次性魔咒的魔力值消耗。



3.传说咒语卡 “火盾护身”



持续时间下调为14秒，同时单个巫师单位处于火墙内部时，3次充能成功所需总时间调整为14秒。



4.史诗咒语卡“闪回咒”



闪回咒语的卡等计算规则修改为与传说咒语卡“标准咒语书”一致，卡牌等级受“闪回咒”和已拥有对应魔咒两者的卡牌等级影响，两者的卡牌等级越高，一次性魔咒的卡牌等级越高。



5.史诗咒语卡“飞来咒”



移除“飞来咒”在决斗俱乐部中“最佳拍档”的专属平衡性调整。
